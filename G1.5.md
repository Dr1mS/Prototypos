## 0. Why this exists (and why it is not the fun part)

G1 proved the **mechanism** (state-injection is the perception bias: care gap
−0.737, pre-committed, care-only so independent of the threat relabel). But three
things are unconfirmed or worrying, and they converge on one existential question:

- P1 (multi-attractor): **not confirmed** — 3 creatures, corner-capture 0 %, all
  near centre.
- P2 (path dependence): **not confirmed AND the reduced figure is bugged** —
  first-quarter valence = 1.00 for all 4 points, r = 0.
- The real model is **systematically more pessimistic than the mock**:
  neutral@damaged −0.75, nurture@damaged **−0.39** (mock only discounted, LLM
  inverts), ablation recovers only to −0.72 (mock → +1). Secure basin is weakly
  fed; damaged basin is strongly fed.

> **The question G1.5 answers: does divergence survive the negativity bias, or
> does everything slide into damaged?** The entire viral hook ("look what mine
> became") rests on variety. If all creatures collapse to one damaged attractor,
> there is no variety, and no archetype detector is worth building yet.

This is decision-critical, not cosmetic. Do it before any viral-artifact work.

## 2. SUBAGENT D (Opus 4.8) — measure the appraisal field  [~180–300 calls]

The appraiser is memoryless given state, so `appraisal(event, state)` is the whole
story. Measure it.

- **Grid.** For each of the 6 event keys × injected A ∈ {−1.0, −0.5, 0, +0.5,
  +1.0} (hold R=0, mood=0, O=0), call `appraise_llm` **N=5×** (deterministic text,
  `text_pick="first"`, temp as in G1). Record mean + stdev of care/threat/novelty/
  autonomy per cell. → `field_A.json` (a 6×5 table with stdevs).
- **R-dependence probe (cheap check).** For the 2 event keys most likely to be
  R-sensitive (`harm`, `scold`), also sweep R ∈ {−1, 0, +1} at A=0, N=5×. If
  threat varies with R by > 0.15, flag that the field needs an R axis (§3 will
  extend); if not, A-only field is sufficient. → note in report.
- **Asymmetry metrics** (compute from the field, put in the report):
  - *neutral fixed point*: the A where `care(neutral, A) = 0` (if none in [−1,1],
    state it — that means neutral always erodes one way).
  - *basin feed asymmetry*: mean care at secure states (A>0) vs at damaged states
    (A<0), across ambiguous events. Quantifies "secure under-fed vs damaged
    over-fed".
  - *pessimism offset*: mean care(field) − mean care(mock) over the shared grid.

Self-test: field table is complete (no missing cell), all stdevs present.

## 3. SUBAGENT E (Opus 4.8) — field-driven divergence + P2 bug fix  [~0 LLM calls]

- **Field appraiser** (`field_appraiser.py`), plugs into the existing seam,
  interpolates the measured field over A, injects measured per-cell noise:

```python
def make_field_appraiser(field, field_std, interp, noise=True):
    # field/field_std: 6x5 tables from field_A.json; interp: linear over A.
    def field_appraiser(ev, s, age, p, bias_on):
        A = s[2]
        care = interp(field[ev], A);  thr = interp(field_thr[ev], A)
        nov  = interp(field_nov[ev], A); aut = interp(field_aut[ev], A)
        if noise:
            care += RNG.normal(0, interp(field_std[ev].care, A)); # etc.
        pv = float(np.clip(care - 0.5*thr + 0.3*aut, -1, 1))   # same adapter math
        pa = float(np.clip(0.7*thr + 0.5*nov, 0, 1))
        return pv, pa, nov
    return field_appraiser
```

## 5. FIELD-FIDELITY SPOT-CHECK — is the field-driven sim faithful?  [~2 500 calls]

The whole cheap-simulation trick assumes the memoryless-field lookup reproduces
real end-to-end runs. Validate it: run a **small real-LLM P1** — 10 creatures ×
250 events through the real `appraise_llm` — and compare their final (A,R) to 10
matched field-driven runs (same seeds/streams). If the two agree (final-state
distributions overlap, no systematic drift), the field method is validated and the
§3 full-N result stands. If they diverge, the memoryless assumption leaks (likely
R- or mood-dependence) — extend the field axis and note it.

Run this **after** §3 so it either confirms or qualifies the full-N result.

## 8. Honest branch

The most likely real outcome, given G1's pessimism numbers, is **"divergence
partially survives, damaged-skewed"** — not a clean 4-corner spread and not a
total collapse. That is fine and expected. The point of G1.5 is to measure *how*
skewed and *what* rescues it, so the design choice (embrace the darkness as the
product, or rebalance for range) is made on numbers, not vibes.
